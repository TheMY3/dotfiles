# Keybindings

## Aerospace

### Focus
| Hotkey | Action |
|--------|--------|
| `alt+h/j/k/l` | Фокус влево/вниз/вверх/вправо |
| `alt+tab` | Переключить между двумя последними воркспейсами |

### Windows
| Hotkey | Action |
|--------|--------|
| `alt+shift+h/j/k/l` | Переместить окно |
| `alt+ctrl+h/j/k/l` | Join — объединить с соседним окном в подконтейнер |
| `alt+t` | Toggle floating / tiling |
| `alt+f` | Fullscreen |
| `alt+shift+f` | Flatten — сбросить layout в ровный ряд |
| `alt+-` | Уменьшить размер |
| `alt+=` | Увеличить размер |

### Layout
| Hotkey | Action |
|--------|--------|
| `alt+/` | Переключить tiles horizontal / vertical |
| `alt+,` | Переключить accordion horizontal / vertical |

### Workspaces
| Hotkey | Action |
|--------|--------|
| `alt+1-9` | Перейти на воркспейс |
| `alt+shift+1-9` | Перенести окно на воркспейс и перейти туда |

### Other
| Hotkey | Action |
|--------|--------|
| `alt+shift+;` | Перезагрузить конфиг |

### Apps
| Hotkey | Action |
|--------|--------|
| `alt+enter` | Открыть новое окно Kitty |
| `alt+e` | Finder |
| `alt+b` | Google Chrome |
